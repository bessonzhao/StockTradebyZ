# B1 选股详细分析

## 1. 选股器概述

B1 选股器是 AgentTrader 项目中的核心选股策略之一，基于 KDJ 分位、知行线和周线多头排列等指标，适合趋势性选股。

## 2. 核心过滤条件

B1 选股器使用了四个核心过滤器，所有过滤器必须同时满足才能入选：

| 过滤器 | 作用 | 参数 |
|--------|------|------|
| KDJQuantileFilter | J 值低位过滤 | j_threshold: J 值绝对阈值<br>j_q_threshold: J 值历史分位阈值<br>kdj_n: KDJ 计算周期 |
| ZXConditionFilter | 知行线条件过滤 | zx_m1-zx_m4: 四条均线参数<br>zxdq_span: 知行线计算周期<br>require_close_gt_long: 收盘价>长期均线<br>require_short_gt_long: 快线>长期均线 |
| WeeklyMABullFilter | 周线多头排列 | wma_short: 短期周均线<br>wma_mid: 中期周均线<br>wma_long: 长期周均线 |
| MaxVolNotBearishFilter | 最大成交量非阴线 | n: 考察周期 |

## 3. 参数计算方式

### 3.1 数据预处理

B1 选股器使用的原始数据未进行前复权处理，直接使用了股票的原始开盘价、最高价、最低价和收盘价。数据预处理步骤包括：

1. 列名统一转换为小写
2. 日期列转换为 datetime 类型
3. 按日期排序
4. 计算换手率指标 `turnover_n`
5. 设置日期为索引

### 3.2 KDJ 指标计算

#### 3.2.1 计算原理

KDJ 指标的计算基于收盘价、最高价和最低价，使用了 Numba 加速计算：

1. 计算 RSV（未成熟随机值）：
   ```python
   low_n  = df["low"].rolling(window=n, min_periods=1).min()
   high_n = df["high"].rolling(window=n, min_periods=1).max()
   rsv    = ((df["close"] - low_n) / (high_n - low_n + 1e-9) * 100).to_numpy()
   ```

2. 使用 Numba 加速的 KDJ 递推计算：
   ```python
   @_njit(cache=True)
   def _kdj_core(rsv: np.ndarray) -> tuple:
       n = len(rsv)
       K = np.empty(n, dtype=np.float64)
       D = np.empty(n, dtype=np.float64)
       K[0] = D[0] = 50.0
       for i in range(1, n):
           K[i] = 2.0 / 3.0 * K[i - 1] + 1.0 / 3.0 * rsv[i]
           D[i] = 2.0 / 3.0 * D[i - 1] + 1.0 / 3.0 * K[i]
       J = 3.0 * K - 2.0 * D
       return K, D, J
   ```

#### 3.2.2 过滤条件

KDJQuantileFilter 的过滤条件为：
```
今日 J < j_threshold  OR  今日 J ≤ 历史累积 j_q_threshold 分位
```

默认参数：
- j_threshold: -5.0
- j_q_threshold: 0.10（即 10% 分位）
- kdj_n: 9

### 3.3 知行线计算

#### 3.3.1 计算原理

知行线包括两条线：

1. 知行短期线（zxdq）：双指数平滑移动平均线
   ```python
   zxdq = close.ewm(span=zxdq_span, adjust=False).mean().ewm(span=zxdq_span, adjust=False).mean()
   ```

2. 知行多空线（zxdkx）：四条均线的平均值
   ```python
   zxdkx = (
       close.rolling(m1, min_periods=m1).mean()
       + close.rolling(m2, min_periods=m2).mean()
       + close.rolling(m3, min_periods=m3).mean()
       + close.rolling(m4, min_periods=m4).mean()
   ) / 4.0
   ```

#### 3.3.2 过滤条件

ZXConditionFilter 的过滤条件为：
```
close > zxdkx（收盘价>长期均线）  AND  zxdq > zxdkx（快线>长期均线）
```

默认参数：
- zx_m1: 10
- zx_m2: 50
- zx_m3: 200
- zx_m4: 300
- zxdq_span: 10
- require_close_gt_long: True
- require_short_gt_long: True

### 3.4 周线多头排列计算

#### 3.4.1 计算原理

1. 计算周线收盘价：
   ```python
   # 按 ISO 年+周分组，取每组最后一个交易日的收盘价
   year_week = close.index.isocalendar().year.astype(str) + "-" + close.index.isocalendar().week.astype(str).str.zfill(2)
   weekly = close.groupby(year_week).last()
   ```

2. 计算周线均线：
   ```python
   ma_s = weekly_close.rolling(s, min_periods=s).mean()
   ma_m = weekly_close.rolling(m, min_periods=m).mean()
   ma_l = weekly_close.rolling(l, min_periods=l).mean()
   ```

3. 判断多头排列：
   ```python
   bull = (ma_s > ma_m) & (ma_m > ma_l)
   ```

#### 3.4.2 过滤条件

WeeklyMABullFilter 的过滤条件为：
```
MA_short > MA_mid > MA_long
```

默认参数：
- wma_short: 10（短期周均线）
- wma_mid: 20（中期周均线）
- wma_long: 30（长期周均线）

### 3.5 最大成交量非阴线计算

#### 3.5.1 计算原理

使用 Numba 加速的滚动窗口计算，找出过去 n 个交易日内成交量最大的那一天，判断是否为阴线：

```python
@_njit(cache=True)
def _max_vol_not_bearish(
    vol: np.ndarray, open_: np.ndarray, close: np.ndarray, n: int,
) -> np.ndarray:
    """滚动 n 日窗口内，成交量最大那天不为阴线（close >= open）。"""
    length = len(vol)
    mask = np.zeros(length, dtype=np.bool_)
    for i in range(length):
        start = max(0, i - n + 1)
        max_v   = vol[start]
        max_idx = start
        for j in range(start + 1, i + 1):
            if vol[j] > max_v:
                max_v   = vol[j]
                max_idx = j
        mask[i] = close[max_idx] >= open_[max_idx]
    return mask
```

#### 3.5.2 过滤条件

MaxVolNotBearishFilter 的过滤条件为：
```
过去 n 个交易日内，成交量最大的那一天不能是阴线（close < open）
```

默认参数：
- n: 20（考察周期）

## 4. 数据处理流程

B1 选股的完整数据处理流程如下：

1. **数据加载**：读取股票日线 CSV 数据
2. **数据清洗**：
   - 列名转换为小写
   - 日期列转换为 datetime 类型
   - 按日期排序
   - 数据切片（根据 start_date、end_date 和 warmup_bars）
3. **换手率计算**：
   ```python
   df["signed_turnover"] = (o + c) / 2 * v
   df["turnover_n"] = df["signed_turnover"].rolling(n_turnover_days, min_periods=1).sum()
   ```
4. **指标预计算**：
   - 计算知行线（zxdq, zxdkx）
   - 计算 KDJ 指标（K, D, J）
   - 计算周线多头排列（wma_bull）
5. **向量化过滤**：使用四个过滤器的 vec_mask 方法进行向量化过滤
6. **生成候选列表**：筛选出所有满足条件的股票

## 6. 前复权处理

从代码分析来看，B1 选股器**没有进行明确的前复权处理**，直接使用了原始的开盘价、最高价、最低价和收盘价数据。这意味着：

- 所有指标计算基于原始价格数据
- 没有考虑分红、送股等因素对价格的影响
- 适合短期趋势分析，但长期分析可能存在偏差

## 7. MACD 指标

B1 选股器**没有使用 MACD 指标**，主要依赖 KDJ、知行线和周线多头排列等指标进行选股。

## 8. 性能优化

B1 选股器采用了多种性能优化技术：

1. **向量化计算**：使用 NumPy 数组进行批量计算，提高效率
2. **Numba 加速**：核心计算函数使用 `@njit` 装饰器编译为机器码
3. **预计算机制**：一次性预计算所有中间指标，避免重复计算
4. **并行化处理**：支持多线程/多进程并行计算
5. **向量化过滤**：使用 vec_mask 方法进行快速过滤

### 9.2 向量化过滤实现

```python
def _apply_vec_filters(df: pd.DataFrame, filters: list) -> np.ndarray:
    """对列表中所有实现了 vec_mask 的 Filter 取交集，返回布尔数组。"""
    mask = np.ones(len(df), dtype=bool)
    for f in filters:
        mask &= f.vec_mask(df)
    return mask
```

## 10. 选股流程

1. **数据准备**：加载股票日线数据，进行清洗和预处理
2. **流动性池构建**：选取换手率前 N 只股票作为流动性池
3. **指标预计算**：对每只股票预计算所有指标
4. **向量化过滤**：使用四个过滤器进行向量化过滤
5. **生成候选列表**：筛选出所有满足条件的股票
6. **去重处理**：合并不同策略的候选列表，去重
7. **输出结果**：将候选股票列表保存到文件

## 11. 结果解读

B1 选股器的候选股票具有以下特征：

- **KDJ 低位**：J 值处于历史低位，有反弹潜力
- **知行线多头**：短期均线和收盘价均在长期均线上方，趋势向上
- **周线多头排列**：周线级别趋势向上
- **量价健康**：成交量最大日非阴线，量价配合良好

这些特征表明，B1 选股器选出的股票具有较强的上涨潜力，适合波段操作。

### 12.1 添加前复权处理

建议在数据预处理阶段添加前复权处理，考虑分红、送股等因素对价格的影响：

```python
def apply_forward_adjustment(df: pd.DataFrame) -> pd.DataFrame:
    """应用前复权处理"""
    # 假设存在复权因子列 adj_factor
    if "adj_factor" in df.columns:
        # 计算累计复权因子
        df["cum_adj_factor"] = df["adj_factor"].cumprod()
        # 应用复权因子
        df["open"] = df["open"] * df["cum_adj_factor"]
        df["high"] = df["high"] * df["cum_adj_factor"]
        df["low"] = df["low"] * df["cum_adj_factor"]
        df["close"] = df["close"] * df["cum_adj_factor"]
    return df
```

```python
def compute_macd(df: pd.DataFrame, fastperiod: int = 12, slowperiod: int = 26, signalperiod: int = 9) -> pd.DataFrame:
    """计算 MACD 指标"""
    exp1 = df["close"].ewm(span=fastperiod, adjust=False).mean()
    exp2 = df["close"].ewm(span=slowperiod, adjust=False).mean()
    macd = exp1 - exp2
    signal = macd.ewm(span=signalperiod, adjust=False).mean()
    histogram = macd - signal
    return df.assign(macd=macd, macd_signal=signal, macd_histogram=histogram)
```
